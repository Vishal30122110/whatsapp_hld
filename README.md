# WhatsApp-like Chat Server (JS)

Minimal scaffold for a WhatsApp-like real-time chat server using Node.js (JavaScript), Express, Socket.IO, MongoDB (Mongoose), and Redis (ioredis).

Quick start:

1. Copy `.env.example` -> `.env` and fill values
2. npm install
3. npm run dev

This repo contains a basic auth flow (JWT), Mongoose models, REST routes, and a simple Socket.IO handler for send_message and presence.
